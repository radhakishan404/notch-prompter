# Installation Guide

This guide covers installation for end users who want to run Notch Prompter without building from source.

## System Requirements

- macOS 13.0 or newer

## Install From DMG

1. Open the latest release page:
   - https://github.com/radhakishan404/notch-prompter/releases/latest
2. Download `NotchPrompter.dmg`.
3. Open the downloaded DMG file.
4. Drag `NotchPrompter.app` into the `Applications` folder.
5. Launch `NotchPrompter` from Applications.

## First Launch Permissions

Notch Prompter uses global keyboard shortcuts. macOS may ask for Accessibility permission.

1. Click `Open System Settings` when prompted.
2. Enable access for `NotchPrompter` in:
   - System Settings -> Privacy & Security -> Accessibility
3. Quit and relaunch the app once permission is enabled.

## Update Process

1. Download the newest DMG from Releases.
2. Replace the existing `NotchPrompter.app` in Applications.
3. Relaunch the app.

## Uninstall

1. Quit Notch Prompter.
2. Move `NotchPrompter.app` from Applications to Trash.
3. Optional: remove user defaults if you want a clean reset.
